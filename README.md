# 最后总结下外卖项目的非常重要的点

1. 要带有裂变功能，实现人拉人，这样才会可持续，用户才会源源不断的流进来。你才能真正意义被动躺赚。

2.要有种子用户，淘宝直通车是一个不错的选择，这样你的用户很快能够起来。这也就是刚开始说为什么淘宝直通车还有机会的原因。

3.不要简单的用公众号直接挂链接，因为微信会封链接，隔一段时间，你的推广链接就变诱导了，这样你的用户全部损失了，还是要挂一个小程序。一定要挂一个小程序。

4.不要脸持续推。
